Description

According to Marijuana Business Daily Legal sales of marijuana are expected to grow to more than $20 billion by 2022, from approximately $10 billion in 2018.

In 2017, $250 million of tax revenue was generated from the sale of marijuana, this has enabled the building of roads and schools in the state of Washington


But it's not all about money!


In Colorado alone, there were almost 3, 000 marijuana sale licences issued to startups across the state. 900 of these operate as dispensaries.
But the incoming prevalence of marijuana can leave people confused and bothered. But at the heart of this fear is a simple lack of knowledge.
We aim to educate the confused and bridge the gap between the stoners in your backyard and confused old people.

We do this by providing users with access to thousands of marijuana strains, as well as their affects. Visitors of our website can register, login to their account, rate strains, and eventually...view local dispensaries. All for free!


Enough reading, let's get you set up.


API Setup


If you're here you probably already have your API set up. However, if not, here's how to do it:
1) Navigate to Evan Busse and register for a key.
2) Clone this project from our Github into a folder on your local machine.
3) Navigate to that folder using $ cd on mac.
4) Run $ mongod to get the mongo database running.
5) In a new tab, but same folder, run $ EVANBUSSE_APIKEY=[Your key you got in step 1] npm start to run the server.
6) Navigate to your favourite browser, (i.e. Chrome), and go to localhost:4200
7) And here we are!
